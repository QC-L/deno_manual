资源 (Resources)§

资源（Resources)，又称 rid，是 Deno 版本的文件描述符。它们是一些整数数值，用来指代打开的文件、套接字 (sockets) 和其他概念。基于 rid，Deno 能够查询系统中有多少个打开的资源，这在测试时很有用。

console.log(Deno.resources());
// { 0: "stdin", 1: "stdout", 2: "stderr" }
Deno.close(0);
console.log(Deno.resources());
// { 1: "stdout", 2: "stderr" }

指标 (Metrics)§

指标 (Metrics) 是 Deno 用于各种统计数据的内部计数器。
